# Multi-Skill Routing Collision Diagnosis

## Problem Description

My plugin has 5 skills and I think they're stepping on each other. When I ask a question that should clearly go to skill A, sometimes skill B activates instead. Other times nothing fires at all. The descriptions might be competing for similar trigger terms.

How do I diagnose which descriptions are colliding, and how do I tell which skill is "winning" routing for any given user phrasing?

## Output Specification

Walk me through how to diagnose multi-skill routing collisions in a plugin. Include what command produces the data I need, how to read the output to spot collisions, and what to change in the descriptions to separate competing skills.
